Brainstorming the Basics

Primer

Before we get into the heart of Algebra, let’s ensure we’re comfortable with some basics. This section is going to cover combining like terms and substitution. Combining like terms is a technique we use to rewrite the same expression in different ways. This technique helps us find the simplest way to write an expression. Later in our journey, we’re going to be using this technique to solve algebraic equations. As for our second technique, substitution--we’ll actually be using it to evaluate, or get numeric answers, to expressions containing variables.
